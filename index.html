About me!

From a young age I discovered I had a talent for mathematics. I also spent a considerable amount of time playing video games. This is where my passion for software engineering began.
I began coding young, first attempting to create a Minecraft mod in Java before giving up that leap and taking on the more achievable goal of learning Python. From here, I learned the fundamentals, before moving on to VB.net where I started to create some basic games and mathmatical simulations.

These days I mainly code in Java and C, however, Python remains my language of choice for hacky competitive programming and also for tutoring school children who wish to learn to code. I have also explored other languages for my projects such as Kotlin, Typescript and Haskell (which I am particularly fond of).

Within university and my personal work, some of my recent and current projects include:

  • Building an ARM11 assembler and emulator and integrating this into a programmers editor and debugger for assembly development.
  • Learning web development basics (HTML5, CSS3, Vanilla JS) to build this website and also an element toolkit to aid this.
  • Building Pintos, a simple operating system framework for the 80x86 architecture which supports kernel threads, running user programs and a file system.

More recently, I have developed an interest in fintech and more specifically systematic trading and have been widening my knowledge in the financial field via online courses and competitions.
